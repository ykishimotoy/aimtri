### 2024-11-30 (6): AI道場の波紋アニメーション表示問題を修正
**問題:**
- AI道場セクションの波紋アニメーションが表示されていなかった
- 原因：z-index の設定ミスで、::before 擬似要素が Canvas の上に重なっていた

**修正内容:**
- .ai-dojo::before の z-index を 1 → 0 に変更
- .ripple-canvas の z-index を 0 → 1 に変更
- これにより波紋が正しく表示されるようになった

**z-index 構造:**
- .ai-dojo::before: z-index: 0（背景グラデーション）
- .ripple-canvas: z-index: 1（波紋アニメーション）
- .dojo-content: z-index: 2（コンテンツ）

**更新ファイル:**
- style.css: z-index の修正
